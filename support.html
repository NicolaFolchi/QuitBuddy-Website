Getting Started

  • Grant Accessibility permission so QuitBuddy can raise apps and send the standard quit command.
  • Grant Input Monitoring permission so the app can detect Cmd+Q presses.
  • Open Preferences → Global Mode to choose your default protection (Confirm, Hold, Double‑Press, or Passthrough).

Per-App Rules

Use the Per-App Rules tab to override the global mode. Each app can have its own combination of Confirm, Hold, Double-Press, or Passthrough. Drag rules to change their priority when overlapping conditions exist.

Export & Import

In Preferences, click “Export Rules” to save a JSON copy of your QuitBuddy settings. To import on another Mac, use “Import Rules” and select the exported file.
